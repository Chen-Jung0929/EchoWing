# BirdCLEF 前端

React + Vite + Tailwind CSS 介面。完整說明（安裝、API、環境變數）請見專案根目錄 [README.md](../README.md)。

## 開發

```bash
npm install
npm run dev
```

需同時啟動後端（預設 `http://127.0.0.1:8000`），開發伺服器會將 `/api` 代理至後端。

## 指令

| 指令 | 說明 |
|------|------|
| `npm run dev` | 開發伺服器 |
| `npm run build` | 正式版建置 |
| `npm run preview` | 預覽建置結果 |
| `npm run lint` | ESLint |
